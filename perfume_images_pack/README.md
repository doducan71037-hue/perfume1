# 香水产品图打包（链接版）

本压缩包里包含：

- `perfume_images.csv`：60条产品清单（含重复项），对应的产品图直链（尽量选了**简单干净**的单品图）。
- `index.html`：一个在线预览画廊（打开需要联网加载图片）。

## 说明

- 大部分图片来自 NoseTime 的产品图 CDN（`img.xssdcdn.com`），少数来自 Fragrantica 的图片主机（`fimgs.net`，多为 **AVIF** 格式）。
- 个别名称在站内只能找到最接近版本：
  - `Hermès Twilly d'Hermès`：用 `Twilly Eau Ginger` 作为接近替代。
  - `Penhaligon's Halfeti`：用 `Halfeti Leather` 作为接近替代。
  - `Penhaligon's Portraits Collection`：用 `Portraits` 系列中的一支（The Bewitching Yasmine）作为代表图。

## 使用与版权

这些图片通常属于品牌/网站的版权内容。若用于商业用途（电商、广告、海报等），建议向品牌/渠道索取授权的官方物料。
